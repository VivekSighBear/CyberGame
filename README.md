# Crack John's Password (Cyber Game) 🎮🔐

A **Streamlit** cyber-themed mini game where players crack a password across **5 levels of hints**.

## Requirements

- Python **3.9+** (recommended: 3.10+)
- `pip`

## Setup

Create and activate a virtual environment (optional but recommended), then install dependencies:

```bash
pip install streamlit
```

## Run

From the project folder (the one containing `app.py`):

```bash
streamlit run app.py
```

The app will open in your browser.

## Project structure

- `app.py`: Streamlit app (UI + game logic)

## Notes

- Password checks are **case-sensitive**.
- UI is styled with custom CSS to achieve a **cyber-game** look and clear visual hierarchy.
